# Free Barcode Generator

This project is a free, browser-based barcode generator that allows users to create various types of barcodes.

## Features

-   **Multiple Barcode Types**: Supports EAN-13, EAN-8, Code 128, Code 39, UPC-A, ITF-14, Codabar, MSI, and Pharmacode.
-   **Single and Batch Generation**: Create one barcode at a time or generate a batch of up to 100 barcodes.
-   **Customization**: Customize barcode colors and background.
-   **Download Options**: Download barcodes as PNG or SVG files. Batch-generated barcodes are downloaded as a ZIP file.
-   **Privacy-Focused**: All processing happens in the browser. No data is sent to any server.

## Usage

1.  Open `index.html` in a web browser.
2.  Select the desired barcode type.
3.  Enter the barcode value or leave it blank to auto-generate a random one.
4.  Click "Generate Barcode".
5.  Download the generated barcode.
